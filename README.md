# The Human Capability Project

The Human Capability Project (THCP) is an initiative focused on understanding, developing, and governing human capability through transparent frameworks, structured observation, and capability-centered systems.

## Current Repository Status

This repository currently contains the foundational governance and charter artifacts for the project.

## Authoritative Documents

### Project Charter

- docs/charters/thcp_project_charter_v1.md

Defines the mission, scope, and purpose of The Human Capability Project.

### PPIP Framework Charter

- docs/charters/ppip_framework_charter_v1.md

Defines the PPIP framework and its role within the project ecosystem.

### Governance Rules
